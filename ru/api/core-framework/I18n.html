Mobicms\i18n

Пакет i18n реализует мультиязычность. Имеется возможность плюрализации множественных чисел. Фразы перевода хранятся в .PO файлах (формат GNU Gettext). Для ускорения работы, применяется кэширование.

Описание

При написании пакета i18n ставилась задача получения функционала, аналогичному GNU Gettext. Это дает много преимуществ, как в удобстве самого кода, так и в использовании многочисленных онлайн-сервисов перевода.

Имеется возможность автоматического определения локали клиента по заголовкам, посылаемым браузером. Если запрашиваемый язык имеется в системе, он будет применен. Дополнительно, у посетителя есть возможность самому выбрать нужный язык. В то же время, Администратор имеет возможность запретить автоматический и пользовательский выбор и принудительно установить нужный язык для всех клиентов.

Для перевода, проект mobiCMS использует онлайн сервис Crowdin. Вы можете свободно принимать участие в переводе, или улучшать имеющийся. Если нужен какой-то другой язык и Вы готовы заняться переводом, регистрируйтесь на Crowdin и сообщите администратору перевода какой язык нужен, по возможности он будет добавлен и Вы сможете заняться переводом.

Использование

Функции перевода

Для упрощения работы с пакетом и уменьшения объема кода, используются функции-псеводнимы, список которых приведен в таблице.

_s($message [, $domain]) Ищет перевод в глобальных системных фразах /system/locale/
Если не обязательный параметр $domain не задан, то по умолчанию ищет в файле default.po.
_sp($singular, $plural, $count [, $domain]) Версия _s() для сообщений во множественном числе
_m($message, [$domain]) Данная функция применяется для перевода фраз модулей. Ищет перевод в папке locales указанного модуля. Если параметр $domain не задан, то по умолчанию ищет в файле default.po.
_mp($singular, $plural, $count, [$domain]) Версия _m() для множественного числа
