API, структура каталогов

Структура каталогов

  • .sources

    В данном каталоге располагаются некоторые исходники (LESS, JS, POT).

    Нужен только для разработчиков, в дистрибутив не входит.
    • mobicms
      LESS файлы для компиляции тем оформления
    • third-party
      Исходники от вендоров (Bootstrap, codemirror, js-cookie, sceditor)
    • translate
      .POT файлы (шаблоны) для локализации системы на другие языки
  • assets
    Файлы, доступные "извне" (то есть, вызываемые браузером) и не относящиеся к конкретным темам оформления.
    • avatars
      Каталог аватаров
    • js
      Общесистемные JS, не относящиеся к конкретной теме оформления (jquery, редакторы).
    • smilies
      Каталог смайлов
  • install
    Инсталлятор системы
  • modules
    Тут располагаются все модули. Данный каталог обрабатывается Роутером, прямой вызов файлов закрыт.
    Модули постоянно меняются и дополняются, поэтому список может быть неполным.
    Структура папок модулей описана в отдельной статье.
    • 404
      Страница 404
    • admin
      Панель администратора
    • community
    • help
      Справочный модуль, аватары и смайлы
    • home
      Домашняя страница
    • login
      Вход на сайт
    • news
      Модуль новостей
    • online
    • profile
    • registration
    • rss
    • whois
  • system

    Ядро системы: Core Framework, конфигурация, кэш, логи, локали, вендоры и др., относящееся к ядру.

    Имейте в виду

    Для того, чтоб не нарушить работоспособность всей системы и не потерять совместимость с последующими обновлениями, не меняйте и не удаляйте никакие файлы, находящиеся в данном каталоге и подкаталогах.

    Исключение составляют папки cache и logs, содержимое которых можно удалять.

    На заметку

    Если в Ваших модулях нужен другой функционал, то используйте свои классы, или наследуйте имеющиеся. Если заметили какую-нибудь ошибку, или есть предложения по усовершенствованию, сообщите на оф. форуме, или пришлите Pull Request в репозиторий.

    • cache

      Системный кэш.

      Содержимое папки можно удалять, на работу системы не влияет.
    • config

      Конфигурационные файлы системы.

      В файле Database.php находятся реквизиты доступа к базе данных.
    • core
      mobiCMS Core Framework
      Структура и подробное описание пакетов находится в отдельных статьях данного раздела..
    • includes
      Различные подключаемые файлы.
    • locale
      Локализация системы (языки).
      Структура папок и подробное описание есть в отдельной статье.
    • logs

      Системные логи.

      Содержимое папки можно удалять, на работу системы не влияет.
    • vendor
      Тут располагаются скрипты вендоров (third-party components).
  • themes
    Темы оформления: шаблоны, CSS, JS, картинки и др.
    Темы постоянно меняются и дополняются, поэтому список может быть неполным.
    • thundercloud

      Тема оформления по-умолчанию.
      Структура папок описана в отдельной статье.

      ВНИМАНИЕ!

      В данной теме находятся файлы, используемые системой и другими темами оформления.
      Поэтому, не меняйте и не удаляйте саму тему, или любые файлы, находящиеся в ней.
  • uploads
    Тут располагаются различные выгружаемые файлы.
